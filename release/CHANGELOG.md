# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [0.2.0] - 2026-01-08

### Added
- Build plugin for Windows and Linux too.

## [0.1.1] - 2026-01-07

### Added
- Support for Skunkcrafts Updater.

## [0.1.0] - 2026-01-07

### Added
- Initial release of the IVAO Joystick PTT plugin.
- Support for mapping joystick buttons to keyboard key presses (default F12).
- Configuration file (`ivao_ptt.toml`) support for custom keys.
- **VATSIM Compatibility:** Option to fall back to `xpilot/ptt` command if enabled.
